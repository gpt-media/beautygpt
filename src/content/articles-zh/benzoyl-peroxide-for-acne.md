如果你长的是红肿、暴躁、带脓的痘痘，想找一种真正对它管用的非处方成分，**过氧化苯甲酰 (benzoyl peroxide) 就是答案**。它能杀死引发炎症性痘痘的细菌，同时给毛孔做个温和的大扫除，这正是皮肤科医生用了几十年的原因。它便宜，在马来西亚任何一家药房都不需要处方就能买到，而且和抗生素不一样，细菌不会对它产生抗药性。

唯一要注意的是，它也是比较容易用错的活性成分之一。一开始就用得太重、太频繁，换来的就是又紧绷、又脱皮、又刺痛的皮肤，在我们这种又热又湿的气候里，这是很实际的风险。所以这篇指南先讲原理：搞懂它*怎么*起作用，正确的用法自然就出来了。

## 过氧化苯甲酰 (benzoyl peroxide) 怎么起作用

炎症性痘痘，就是那种又红、又肿、有时还会痛的，主要是由一种叫痤疮丙酸杆菌 (Cutibacterium acnes) 的细菌引起的（你可能还会看到它以前的名字，痤疮丙酸杆菌 Propionibacterium acnes）。它在堵塞、充满油脂、缺氧的毛孔里活得很自在，不断繁殖，引发免疫反应，把原本安静的堵塞毛孔变成又红又痛的痘痘。

过氧化苯甲酰 (benzoyl peroxide) 攻击它的方式很巧妙。涂到皮肤上后，它会分解，**往毛孔里释放氧气**。那些细菌没法在氧气环境里好好生存，数量就大幅减少。正如 [DermNet](https://dermnetnz.org/topics/benzoyl-peroxide) 所解释的，它能减少皮肤表面的细菌数量，又不会助长抗生素抗药性。说白了，它就是一种不靠抗生素的抗菌成分。

这一个机制就给了它两个实际的优势：

- **不产生抗药性。** 细菌会随着时间对抗生素演化出抗药性，但它们没办法真正适应被氧化，所以过氧化苯甲酰 (benzoyl peroxide) 年复一年地都管用。这也是为什么医生常常把它*和*外用抗生素搭配着开：它能保护抗生素，避免养出抗药性的细菌。
- **温和的去角质作用。** 除了抗菌，它还能帮忙代谢死皮、清掉一些堵在毛孔里的油脂和杂质。它不是强力去角质成分，但能帮毛孔往疏通的方向走一点。

它*不太*管用的地方：纯粹没有炎症的黑头和白头，以及那种深层、囊肿型、长在皮肤底下的痘痘。对于堵塞但不发炎的毛孔，通常用另一种活性成分更合适，下面会讲到。

> 经验法则：如果痘痘是红的、鼓起来的、发炎的，那就最适合用过氧化苯甲酰 (benzoyl peroxide)。如果是扁平的黑头，或者皮肤底下一个安静的小鼓包，就该换别的了。

## 它最适合对付什么

痘痘不是只有一种，过氧化苯甲酰 (benzoyl peroxide) 也不是万能药。把成分和痘痘类型配对好，本身就是成功的一大半。

| 痘痘类型 | 过氧化苯甲酰 (benzoyl peroxide) 适合吗？ |
|---|---|
| 丘疹（小的红色鼓包） | 适合，很好的选择 |
| 脓疱（红色鼓包，顶上有白色脓头） | 适合，很好的选择 |
| 轻度到中度的炎症性痘痘，总体来说 | 适合，一线选择 |
| 黑头和白头（粉刺） | 只适合一部分；水杨酸 (salicylic acid) 或类视黄醇 (retinoid) 通常更管用 |
| 囊肿型／结节型／皮下痘痘 | 不适合，这需要看医生，不是非处方药膏能搞定的 |
| 下巴沿线的荷尔蒙痘痘 | 对发炎的痘痘有帮助，但没法处理背后的荷尔蒙原因 |

简单说，过氧化苯甲酰 (benzoyl peroxide) 是对付**轻度到中度炎症性痘痘**的非处方主力。它在大多数人天天都要面对的那一颗颗红痘痘上表现出色，背后有几十年的证据支撑。

## 选浓度：比你想的要低

关于这个成分，最有用、也最反直觉的一个事实是：**用得多不代表更好。**

非处方的过氧化苯甲酰 (benzoyl peroxide) 浓度通常大致在 **2.5% 到 10%** 之间。很多人下意识就会拿那个 10% 的，想说效果"更强"。但这件事已经被反复研究过，一致的结论是：**2.5% 清痘的效果和 5% 或 10% 差不多，但皮肤干燥、发红和刺痛却少很多。** 浓度更高，大多只是给你换来更多刺激，而不是更多效果，这也是为什么[美国皮肤病学会](https://www.aad.org/public/diseases/acne)把低浓度的过氧化苯甲酰 (benzoyl peroxide) 列为首选的非处方选项之一。

所以对几乎所有人来说，明智的起点都是 **2.5% 或 5%**。更高浓度留给你已经确认能耐受的皮肤。不管你选哪个，都以标签上的说明为准，那反映的是这个配方经过测试的正确用法。

过氧化苯甲酰 (benzoyl peroxide) 还有好几种剂型，而在潮湿的气候里，剂型和浓度一样重要：

- **洁面／洗剂。** 抹上去、停留一小段时间后冲掉。比较温和，因为它在皮肤上停留的时间短，对敏感肌或第一次用的人来说是个不错的入门选择，用在背部和胸口也很方便。
- **停留型凝胶或乳霜。** 因为留在皮肤上，所以更强效。就是经典的"点涂在痘痘上"，或者全脸薄薄涂一层。
- **复方处方。** 医生可以开把过氧化苯甲酰 (benzoyl peroxide) 和类视黄醇 (retinoid)（比如阿达帕林 adapalene）或外用抗生素做在同一个产品里的处方药。

## 怎么引入它而不毁掉皮肤

这是大多数人出错的地方。过氧化苯甲酰 (benzoyl peroxide) 很有效，但要是从第一天就用得又重又勤，它会让皮肤变干、损伤皮肤屏障，而在马来西亚的高温潮湿里，受损的屏障会脱皮、上防晒时刺痛，甚至可能让你的皮肤*看起来更糟*。解决办法就是从低浓度、低频率开始，慢慢来。

温和的入门流程是这样的：

1. **先做斑贴测试。** 在一小块区域（前臂内侧或下巴的一小块地方）涂一点，连续两三天，看看你会不会有不良反应。
2. **一开始一周用几次，别天天用。** 从隔一两个晚上一次开始，涂在洁净、干燥的皮肤上。
3. **薄薄涂一层。** 全脸用黄豆大小的量，或者在单颗痘痘上点一小点。涂得厚不会更快见效，只会更刺激。
4. **做好保湿。** 接着用一款简单、不致痘的保湿霜来抵消干燥。在我们这种气候里，这正是让屏障保持完好、好让你能继续用活性成分的关键。
5. **慢慢加量。** 如果用了一两周后皮肤感觉舒服，就在标签允许的范围内，逐步加到每天一次或一天两次。

要有个短暂的适应期的心理准备。早期有点干、偶尔掉点皮是正常的；但持续的灼烧感、发红刺痛或肿胀就不正常了。那是在提醒你要减少频率或降低浓度，或者停用。

## 没人提、等到出事才知道的布料漂白警告

这一点值得单独开一个标题，因为它几乎让所有人都措手不及。**过氧化苯甲酰 (benzoyl peroxide) 是一种氧化剂，会漂白有颜色的布料，而且是永久性的。**

那个深蓝色的枕套、你最爱的彩色毛巾、深色衬衫的领口：只要过氧化苯甲酰 (benzoyl peroxide) 在皮肤上还没干的时候碰到它们，任何一样都会沾上浅色或橙色的斑块。它在夜里悄悄发生，等你发现的时候，已经无法挽回了。

要避免这种情况：

- 用它的期间，换成**白色纯棉**的枕套和毛巾。
- 让产品在皮肤上**完全吸收、干透**之后，再让皮肤碰到任何布料，包括睡前。
- 涂完后把手**彻底冲洗**干净。
- 小心衣领、围巾，还有你擦脸用的那条毛巾。

## 把过氧化苯甲酰 (benzoyl peroxide) 和其他活性成分搭配

大多数人不会单用过氧化苯甲酰 (benzoyl peroxide)，所以怎么和其他成分错开时间就很重要。

- **和水杨酸 (salicylic acid)。** 两者都没问题，这个搭配也很常见（过氧化苯甲酰用在发炎的痘痘上，水杨酸用在堵塞的地方）。但两者都让皮肤干，所以别在同一周开始用。先引入一种，让皮肤稳定下来，再加另一种。
- **和类视黄醇 (retinoid)（阿达帕林 adapalene、维 A 酸 tretinoin）。** 要小心。同时用，这两者可能让皮肤过度受刺激，而且过氧化苯甲酰 (benzoyl peroxide) 接触到某些较老的类视黄醇时会把它化学性地破坏掉。常见的变通办法是把它们分开：早上用过氧化苯甲酰 (benzoyl peroxide)，晚上用类视黄醇 (retinoid)。（有些较新的配方经过稳定化处理，可以一起用，但要看标签。）我们这篇[马来西亚的阿达帕林 (adapalene)](/my/articles/adapalene-malaysia/) 指南，讲了怎么安全地叠加它。
- **和烟酰胺 (niacinamide)。** 这两个很搭：烟酰胺能帮忙舒缓发红、支撑屏障，抵消一部分干燥。
- **和维生素 C (vitamin C)。** 过氧化苯甲酰 (benzoyl peroxide) 会氧化某些形式的维生素 C，削弱它的效果。把它们放在不同的护肤流程里（一个早上、一个晚上）。
- **一定要搭配防晒。** 过氧化苯甲酰 (benzoyl peroxide) 会让皮肤变得稍微敏感一点，而一边治痘痘、一边又在累积新的晒伤和黑印，根本是白忙一场。在马来西亚的太阳下，每天涂防晒是绝对不能省的。

## 马来西亚买得到什么，大概多少钱

过氧化苯甲酰 (benzoyl peroxide) 在这里很好找，实体店和网上都有。价格只是大概，请以官方店铺标价为准。

- **Oxy** 是很多马来西亚人从小就认识的老牌药用品牌，在药房以点涂祛痘产品和洗剂的形式出售，大概 RM15–30。是个简单、好入手的入门选择。
- **Benzac／过氧化苯甲酰 (benzoyl peroxide) 凝胶**（常见 2.5% 或 5%）在 **Guardian 和 Watsons** 普遍有货，通常在 RM20–40 左右。如果在开放货架上没看到，可以问问药剂师，因为有些是放在柜台后面的。
- **PanOxyl** 是一款很受欢迎的过氧化苯甲酰 (benzoyl peroxide) 洗剂，在 **Shopee、Lazada 和 TikTok Shop** 上作为进口商品出现，通常在 RM40–70 左右。请向信誉好的卖家购买，并检查保质期。
- **药剂师推荐的本地同类产品**往往最划算，因为很多马来西亚药房卖的通用版过氧化苯甲酰 (benzoyl peroxide) 比进口品牌便宜。值得开口问一问。

因为过氧化苯甲酰 (benzoyl peroxide) 会随着时间和高温而降解，一支便宜、运了很久、又快过期的，其实是省了小钱、亏了大钱。这东西本来就不贵，去本地药房买新鲜的，通常才是更聪明的做法。

## 什么时候该跳过它、去看专业人士

过氧化苯甲酰 (benzoyl peroxide) 治的是普通的、轻度到中度的炎症性痘痘。它在这方面是真的好，在别的方面也是真的不行。如果你的痘痘**严重、深层疼痛、囊肿型、在留疤，或者坚持用了两个月也根本没改善**，那不是要你去换一支更强的药膏的信号。那是该去看医生或药剂师的信号。处方选项（口服药、处方强度的类视黄醇 retinoid、荷尔蒙疗法）的存在，正是为了对付那些药膏处理不了的痘痘，而越早开始用，越能保护你免于留疤。

这篇文章是科普性质的，不是医疗建议。关于处方治疗，或者严重、疼痛或反复的痘痘，请去看医生或药剂师。

## 总结

过氧化苯甲酰 (benzoyl peroxide) 是对付红肿发炎痘痘最好的非处方成分，因为它能杀死背后的细菌：从 2.5% 或 5% 开始，慢慢来，做好保湿，别让它碰到你的彩色毛巾，给它四到八周再下判断。
