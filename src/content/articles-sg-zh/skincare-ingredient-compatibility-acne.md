想要简短版的话：**大多数祛痘成分叠在一起都没问题。真正要注意的就那么几条。第一，别一次堆好几种强效活性成分，因为毁掉大多数护肤步骤的是刺激，不是化学。第二，别把纯过氧化苯甲酰 (benzoyl peroxide) 和视黄醇 (retinol) 这类普通 retinoid 叠在同一层，因为过氧化苯甲酰会让那个 retinoid 分解（阿达帕林 (adapalene) 是例外，它和它在一起很稳定）。那条出名的「绝不能把烟酰胺和酸一起用」是个迷思。**

「我能不能把 X 和 Y 一起用」是整个护肤领域里被搜得最多的问题之一，而网上满是看起来很吓人、却大多是错的搭配图表。所以我们来一条条看真正重要的组合，连同每一条背后的机制，让你能搭出一套有效的步骤，而不是一套只会刺激皮肤的步骤。

## 最重要的一条规则：别用太多

在谈任何具体搭配之前，最大的错误是同时用太多强效活性成分。皮肤又红、又刺、又脱皮、突然变得更敏感，远远更多是因为步骤超载，而不是因为某个「不兼容」的组合。

> 诚实的重点：毁掉大多数祛痘步骤的是刺激，不是化学反应。挑一种主力成分，把它撑起来，任何新东西都一样一样加。

一套可行的祛痘步骤通常只需要一种主力成分，也就是[水杨酸 (salicylic acid)](/sg/zh/articles/salicylic-acid-for-acne/)、[过氧化苯甲酰 (benzoyl peroxide)](/sg/zh/articles/benzoyl-peroxide-for-acne/)，或者像[阿达帕林 (adapalene)](/sg/zh/articles/adapalene-singapore/) 这样的 retinoid，再加上温和的配角（洁面、烟酰胺、乳液和防晒）。当你引入一种新的活性成分时，给它几周时间再加下一种，这样万一你的皮肤起反应，你就知道是谁。这和[爆痘期 (purging) 与普通长痘](/sg/zh/articles/acne-purging-vs-breaking-out/) 背后是同一个道理：一种新成分在稳定下来之前可能会先把状况搅一搅，而如果你一次改了五样东西，你根本看不出发生了什么。

## 那些「确实」值得在意的组合

有几个搭配是有真实依据的。没有一个是「危险」的，它们关乎时机，以及怎么从每种成分里拿到全部价值。

### 过氧化苯甲酰 + 普通视黄醇或维 A 酸：把它们分开

这是祛痘护理里唯一一个真实的兼容性注意事项。过氧化苯甲酰 (benzoyl peroxide) 是强氧化剂，当它和普通视黄醇 (retinol) 或维 A 酸 (tretinoin) 处在同一层、同一时间时，会把那个 retinoid 氧化、部分分解，让你拿到一个效力更弱的产品。简单的解决办法：在不同时间用，经典做法是早上用过氧化苯甲酰、晚上用 retinoid。

**重要的例外是阿达帕林 (adapalene)。** 阿达帕林是更稳定的现代 retinoid，研究已经显示[过氧化苯甲酰不会明显让它分解](https://pmc.ncbi.nlm.nih.gov/articles/PMC2958193/)。正是这种稳定性，让阿达帕林和过氧化苯甲酰被做成单一固定复方一起卖。如果你想同时用一种 retinoid 和过氧化苯甲酰，阿达帕林就是那个最好相处的 retinoid。关于 retinoid 家族的更多内容，见[视黄醇与 retinoid 对付痘痘](/sg/zh/articles/retinol-vs-retinoids-for-acne/)。

### 水杨酸（或果酸）+ retinoid：温和点，别硬叠

像水杨酸 (salicylic acid)（或果酸 AHA）这样的 BHA 和 retinoid 之间没有有害反应。问题纯粹是叠加性的刺激：两者都会加快细胞代谢，所以每天晚上把两者的强效版本叠在同一层，常常会带来干燥、脱皮和受压的屏障。聪明的做法是隔晚交替，或者一个早上、一个晚上，并且始终配乳液。如果你在去角质的酸之间做选择，[BHA 与 AHA 对付痘痘](/sg/zh/articles/bha-vs-aha-for-acne/) 解释了哪种适合哪种皮肤。

### 维生素 C + 过氧化苯甲酰：分开更好

当维生素 C 和过氧化苯甲酰一起涂时，过氧化苯甲酰 (benzoyl peroxide) 会降低某些纯维生素 C (vitamin C，即 ascorbic acid) 的效力，这同样是氧化问题，不是危险。简单的答案是把它们放在一天的两端：维生素 C 早上、过氧化苯甲酰晚上，或者隔天用。如果一款产品本身就把两者配在一起卖，配方师已经把这对组合稳定好了，那就照说明用。

## 那些其实是迷思的「不兼容」

网上很多吓人的图表，重复的都是一些对现实产品并不成立的说法。

- **烟酰胺 + 酸或维生素 C。** 这是最顽固的迷思。[烟酰胺 (niacinamide)](/sg/zh/articles/niacinamide-for-acne-and-marks/) 和水杨酸 (salicylic acid)、果酸 (AHA)、维生素 C (vitamin C)、retinoid 和过氧化苯甲酰 (benzoyl peroxide) 都能很好地叠加。那个旧的「烟酰胺抵消维生素 C」的说法，来自高温和实验室条件，并不适用于你放在室温下的护肤成品。烟酰胺是最好配合的活性成分之一。
- **烟酰胺 + 视黄醇「必须」分开。** 不必。两者经常被配在一起，恰恰是因为烟酰胺能缓和 retinoid 的刺激。
- **一套步骤里「绝不能」用两种活性成分。** 可以用。上限是你皮肤的耐受度，不是某本规则手册。很多好的步骤会搭配，比如早上一个抗氧化剂、晚上一个 retinoid。

这些迷思之所以传播，是因为「别混用」比「这取决于你的皮肤和配方」更容易转发。但对你真正会买的产品来说，兼容性很少是问题所在。

## 一套有效的简单叠加顺序

当你确实要叠加时，可靠的原则是在干净的皮肤上由稀到稠：

| 步骤 | 这一步放什么 | 备注 |
|---|---|---|
| 1 | 洁面 | 先把皮肤洗干净 |
| 2 | 水基的治疗 / 精华 | 例如水杨酸 (salicylic acid) 水、烟酰胺 (niacinamide)、维生素 C（早上） |
| 3 | 更稠的治疗乳霜 | 过氧化苯甲酰 (benzoyl peroxide) 或 retinoid，按说明用；不一定要同时用两者 |
| 4 | 乳液 | 缓冲活性成分，支撑屏障 |
| 5 | 防晒（只在早上） | 不能省，尤其是用了去角质的酸和 retinoid 时 |

你不需要每天都做每一步。一个常见、低刺激的模式是：活性成分晚上用，抗氧化剂和防晒早上用，每周再留一两个晚上让皮肤休息。想要完整的、以机制为先的搭建，见[如何在新加坡治疗痘痘](/sg/zh/articles/how-to-treat-acne-singapore/) 和[如何在新加坡建立祛痘护肤步骤](/sg/zh/articles/how-to-build-acne-skincare-routine-singapore/)。

## 关于敏感肌和气候的一点说明

在新加坡的高温潮湿里，皮肤可能感觉更油，底下却仍然在受刺激，所以「成分越多 = 见效越快」的直觉经常适得其反。如果你皮肤敏感，多依靠更温和的成分，把它们错开用，永远别省乳液。如果你叠了好几样强效的东西、皮肤却不开心，解决办法几乎总是做减法，而不是加法。

> 本文为科普内容，并非医疗建议。如需处方治疗，或针对严重、疼痛或反复发作的痘痘，请咨询医生或药剂师。

**结论：** 大多数祛痘活性成分混用都没问题。把过氧化苯甲酰和普通视黄醇或维 A 酸放在不同的时段（阿达帕林是那个稳定的例外），把维生素 C 和过氧化苯甲酰分开，叠加酸和 retinoid 时温和一点。别理那些烟酰胺的迷思。最重要的是，用比你以为的更少的强效活性成分，一样一样地引入，在干净的皮肤上由稀到稠地叠加，并且每天早上都涂防晒。
